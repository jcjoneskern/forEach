How to forEach

The first thing you need for a forEach loop is an array to use it on. Let's use this one here.
var cats = ["siamese", "persian", "manx", "maine coon", "tabby"];
The next thing to understand is that forEach is a method that can only be used on arrays. As with other methods, the way you utilize it is with dot notation (because it's a method) and parentheses (because methods are functions and you need parentheses to call functions).
cats.forEach(); var cats = ["siamese", "persian", "manx", "maine coon", "tabby"];
However, this isn't enough for forEach. This method takes a function as an argument. In other words, we need to put a function inside those parentheses.
cats.forEach(function() { }); var cats = ["siamese", "persian", "manx", "maine coon", "tabby"];
This might be starting to look kinda funky, but let's add one more layer. The function within the forEach loop also requires a parameter. This parameter will represent each item in the array as the loop iterates. You can name it whatever you want, but standard practice is to give you array a plural name (like "cats") and give your parameter the corresponding singular name (like "cats").
cats.forEach(function(cat) { }); var cats = ["siamese", "persian", "manx", "maine coon", "tabby"];
The reason why we generally do things this way is because for every iteration of our forEach loop, the parameter we name in our function will represent each individual item in the array. Anything we write in the code block is what we want to have happen to each item in the array. Let's start simple.
cats.forEach(function(cat) { console.log(cats); }); var cats = ["siamese", "persian", "manx", "maine coon", "tabby"];
